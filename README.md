# 自行車道地圖資訊整合網 | [Demo](https://johnsonmao.github.io/taiwan_bus)

## 作品說明

The F2E 全台公車動態時刻查詢應用服務

## 系統說明

本專案運行方式是使用 `yarn`、`yarn start`

## 資料夾說明

```
|- /src
    |- /api：存放 API 串接整合資料
    |- /asset：通用 Icon 與 SCSS
    |- /components：元件
    |- /pages：頁面
    |- /utils：通用工具、設置
```

## 使用技術

- React
- Leaflet
- PubSub-js
- Bootstrap 5
